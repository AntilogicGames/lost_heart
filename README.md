# Lost Heart

### A Just-click-game. Try to drag your heart to any blackhole available. The closer to the center, more points for you!


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
